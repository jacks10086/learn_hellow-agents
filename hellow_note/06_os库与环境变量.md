# os 库与环境变量

## 一、os 库概述

**os 是 Python 标准库，用于与操作系统交互。**

---

## 二、环境变量操作

### 获取环境变量
```python
import os

# 获取环境变量，不存在返回 None
api_key = os.getenv("API_KEY")

# 获取环境变量，不存在返回默认值
api_key = os.getenv("API_KEY", "default_value")

# 另一种写法
api_key = os.environ.get("API_KEY")

# 判断是否存在
if "API_KEY" in os.environ:
    print("API_KEY 已设置")
```

### 设置环境变量
```python
import os

# 设置
os.environ["MY_VAR"] = "hello"

# 获取
print(os.environ["MY_VAR"])  # "hello"
```

## 三、dotenv 库

### 作用
从 `.env` 文件加载环境变量到 `os.environ`。

### .env 文件
```bash
# .env 文件
API_KEY=sk-xxxxx
BASE_URL=https://api.example.com
MODEL=gpt-4
SERPAPI_API_KEY=your_key_here
```

### 使用
```python
from dotenv import load_dotenv
import os

# 加载 .env 文件
load_dotenv()

# 然后就可以用 os.getenv() 获取
api_key = os.getenv("API_KEY")
```

### load_dotenv 查找顺序
1. 当前工作目录的 `.env`
2. 向上查找父目录
3. 可以指定路径：`load_dotenv(".env.prod")`

## 四、文件和目录操作

### 当前目录
```python
os.getcwd()  # 获取当前工作目录
os.chdir("/path")  # 切换目录
```

### 创建和删除
```python
os.mkdir("dir")        # 创建目录
os.makedirs("a/b/c")   # 创建多层目录
os.remove("file.txt")  # 删除文件
os.rmdir("dir")        # 删除空目录
```

### 列出目录
```python
os.listdir(".")  # 返回当前目录文件列表
```

### 判断存在
```python
os.path.exists("file.txt")   # 文件或目录是否存在
os.path.isfile("file.txt")   # 是否是文件
os.path.isdir("dir")         # 是否是目录
```

## 五、路径操作

```python
import os

# 拼接路径（跨平台）
path = os.path.join("folder", "subfolder", "file.txt")
# Windows: 'folder\\subfolder\\file.txt'
# Linux: 'folder/subfolder/file.txt'

# 获取文件名
os.path.basename("/path/to/file.txt")  # 'file.txt'

# 获取目录名
os.path.dirname("/path/to/file.txt")   # '/path/to'

# 获取扩展名
os.path.splitext("file.txt")  # ('file', '.txt')
```

## 六、为什么用环境变量？

### 问题：硬编码
```python
# ❌ 不安全
api_key = "sk-xxxxx"  # 直接写在代码里
```

### 解决：环境变量
```python
# ✅ 安全
api_key = os.getenv("API_KEY")  # 从环境变量读取
```

### 好处
1. **安全**：敏感信息不暴露在代码中
2. **灵活**：不同环境用不同配置
3. **版本控制**：`.env` 加入 `.gitignore`，不上传
