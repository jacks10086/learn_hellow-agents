# Python 模块与导入机制

## 一、import 执行时机

**Python 导入模块时，会从上到下执行模块里的所有顶级代码。**

### 示例

**mymath.py**
```python
print("① mymath 开始加载")

def add(a, b):
    return a + b

print("② mymath 定义完 add 函数")

x = add(1, 2)
print(f"③ mymath 计算结果: {x}")
```

**main.py**
```python
print("A. main 开始")

import mymath  # 导入时执行 mymath.py 里的所有代码

print("B. main 导入完成")
```

### 运行结果
```
A. main 开始
① mymath 开始加载
② mymath 定义完 add 函数
③ mymath 计算结果: 3
B. main 导入完成
```

## 二、`__name__` 变量

### `__name__` 的值
| 场景 | `__name__` 的值 |
|------|-----------------|
| 文件被**直接运行** | `"__main__"` |
| 文件被**导入** | 模块名（文件名） |

### 示例

**mymath.py**
```python
print(f"mymath 的 __name__ = {__name__}")

if __name__ == "__main__":
    print("直接运行 mymath.py")
```

**main.py**
```python
import mymath
print(f"main 的 __name__ = {__name__}")
```

### 运行结果
```bash
$ python mymath.py
mymath 的 __name__ = __main__
直接运行 mymath.py

$ python main.py
mymath 的 __name__ = mymath
main 的 __name__ = __main__
```

## 三、`if __name__ == '__main__':` 的作用

### 问题：没有保护

```python
# mymath.py
def add(a, b):
    return a + b

# 测试代码（没有保护）
print("测试:", add(1, 2))
```

```python
# main.py
import mymath  # 会打印 "测试: 3"，但我们不想要这个！
```

### 解决：加上保护

```python
# mymath.py
def add(a, b):
    return a + b

# 测试代码（有保护）
if __name__ == "__main__":
    print("测试:", add(1, 2))
```

```python
# main.py
import mymath  # 不会打印测试代码了
```

## 四、模块导入后不会重新执行

```python
import mymath  # 第一次导入，执行 mymath.py
import mymath  # 第二次导入，不执行（已缓存）
```

**Python 会缓存已导入的模块在 `sys.modules` 中。**

## 五、不要随意修改 sys.stdout

### 问题代码
```python
import sys
import io
sys.stdout = io.TextIOWrapper(sys.stdout.buffer, encoding='utf-8')
```

### 问题原因
```
程序启动
    │
    ▼
sys.stdout = io.TextIOWrapper(...)
    │
    ▼
模块执行完毕
    │
    ▼
TextIOWrapper 可能被垃圾回收
    │
    ▼
回收时调用 close()
    │
    ▼
sys.stdout.buffer 被关闭
    │
    ▼
后续 print() 报错！
```

### 正确做法
- 用环境变量 `PYTHONIOENCODING=utf-8` 解决编码问题
- 不要在代码里修改 `sys.stdout`

## 六、from ... import ... 语法

### 导入整个模块
```python
import mymath
mymath.add(1, 2)
```

### 导入特定函数
```python
from mymath import add
add(1, 2)  # 不需要 mymath.
```

### 导入并重命名
```python
from mymath import add as my_add
my_add(1, 2)

import numpy as np
np.array([1, 2, 3])
```

## 八、C++ 对照

| Python | C++ |
|--------|-----|
| `import mymath` | `#include "mymath.h"` |
| 导入时执行代码 | include 只是复制声明 |
| `__name__` 判断 | 无需（没有导入执行机制） |
