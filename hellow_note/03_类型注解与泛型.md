# Python 类型注解与泛型

## 一、类型注解基础

### 函数参数和返回值
```python
def add(a: int, b: int) -> int:
    return a + b
```

### 变量注解
```python
name: str = "张三"
age: int = 25
scores: list[int] = [90, 85, 95]
```

---

## 二、类型注解是可选的

```python
# 无类型注解 - 合法
def add(a, b):
    return a + b

# 有类型注解 - 更清晰
def add(a: int, b: int) -> int:
    return a + b

# 类型注解只是"提示"，运行时不强制
def add(a: int, b: int) -> int:
    return a + b

add("hello", "world")  # 运行正常！返回 "helloworld"
```

## 三、typing 模块常用类型

```python
from typing import List, Dict, Tuple, Optional, Any, Union, Callable

# 列表
names: List[str] = ["张三", "李四"]

# 字典
ages: Dict[str, int] = {"张三": 25}

# 元组
point: Tuple[int, int] = (10, 20)

# 可选类型（可能是 None）
name: Optional[str] = None  # 等价于 str | None

# 任意类型
data: Any = "anything"

# 联合类型
id: Union[int, str] = 123  # 等价于 int | str

# 可调用对象
func: Callable[[int, int], int]  # 接收两个 int，返回 int
```

---

## 四、Python 3.9+ 新语法

```python
# 不需要导入 typing，直接用内置类型
names: list[str] = ["张三", "李四"]
ages: dict[str, int] = {"张三": 25}

# Python 3.10+ 联合类型
def func(x: int | str) -> int | None:
    pass
```

## 五、泛型概念

### 什么是泛型？
**泛型 = 参数化类型**，让"类型"也变成参数。

### 内置泛型类型
```python
# List[元素类型]
List[int]           # 整数列表
List[str]           # 字符串列表

# Dict[键类型, 值类型]
Dict[str, int]      # 键是 str，值是 int

# 嵌套泛型
List[Dict[str, str]]  # 列表里放字典
```

## 六、类型注解 vs 创建数据

| 用途 | 语法 | 示例 |
|------|------|------|
| 类型注解 | `[]` 方括号 | `List[int]`, `Dict[str, str]` |
| 创建数据 | `{}` 大括号 | `[1, 2, 3]`, `{"a": 1}` |

```python
# 类型注解
messages: List[Dict[str, str]]

# 创建数据
messages = [
    {"role": "system", "content": "..."},
    {"role": "user", "content": "..."}
]
```

## 七、Any 类型

### Any 表示任意类型
```python
from typing import Any

data: Any = 123
data = "hello"
data = [1, 2, 3]
data = {"name": "张三"}
```

### Any vs object
```python
# object：严格检查
def process1(obj: object):
    obj.upper()  # ❌ 类型检查报错

# Any：不做检查
def process2(data: Any):
    data.upper()  # ✅ 类型检查不报错（运行时可能报错）
```

### 什么时候用 Any？
- 类型不确定
- 接口返回值类型不确定
- 混合类型容器

## 八、Callable 类型

```python
from typing import Callable

# 简单写法：任何可调用对象
func: Callable

# 完整写法：指定参数和返回类型
# Callable[[参数类型], 返回类型]
my_func: Callable[[int, int], int]
```

### 示例
```python
def register_tool(func: Callable) -> None:
    """注册工具函数"""
    func()
```

## 九、C++ 对照

| Python | C++ |
|--------|-----|
| `List[int]` | `std::vector<int>` |
| `Dict[str, int]` | `std::map<std::string, int>` |
| `Optional[str]` | `std::optional<std::string>` |
| `Any` | `std::any` |
| `Callable[[int], str]` | `std::function<std::string(int)>` |
| 泛型语法 `[]` | 模板语法 `<>` |

## 十、类型注解的好处

1. **代码可读性** - 一眼看出参数和返回值类型
2. **IDE 支持** - 自动补全、参数提示
3. **静态检查** - mypy 等工具检查类型错误
4. **文档作用** - 类型注解就是最好的文档
