# Python 异常处理

## 一、try-except 基本语法

```python
try:
    # 可能出错的代码
except 错误类型:
    # 出错后执行的代码
```

---

## 二、基本示例

```python
try:
    result = 10 / 0
except ZeroDivisionError:
    print("除数不能为零")
```

---

## 三、捕获异常详情

```python
try:
    result = 10 / 0
except ZeroDivisionError as e:
    print(f"错误详情: {e}")
```

---

## 四、捕获多种异常

### 方式1：用元组
```python
try:
    # 代码
except (ValueError, TypeError) as e:
    print(f"错误: {e}")
```

### 方式2：多个 except 块
```python
try:
    # 代码
except ValueError:
    print("值错误")
except TypeError:
    print("类型错误")
except Exception as e:
    print(f"其他错误: {e}")
```

---

## 五、完整结构

```python
try:
    result = risky_operation()
except ValueError as e:
    print(f"值错误: {e}")
except TypeError as e:
    print(f"类型错误: {e}")
else:
    # 没有异常时执行
    print(f"成功: {result}")
finally:
    # 无论如何都执行
    print("清理资源")
```

---

## 六、常见异常类型

| 异常 | 说明 | 触发场景 |
|------|------|----------|
| `Exception` | 所有异常的基类 | 捕获所有异常 |
| `ValueError` | 值错误 | `int("abc")` |
| `TypeError` | 类型错误 | `"a" + 1` |
| `KeyError` | 键不存在 | `dict["不存在"]` |
| `IndexError` | 索引越界 | `list[100]` |
| `ZeroDivisionError` | 除零错误 | `1 / 0` |
| `FileNotFoundError` | 文件不存在 | `open("不存在.txt")` |

## 七、项目中的例子

```python
# tools.py 中的天气 API 错误处理
try:
    response = requests.get(url)
    response.raise_for_status()
    data = response.json()
except requests.exceptions.RequestException as e:
    return f"错误：查询天气时遇到网络问题 - {e}"
except (KeyError, IndexError) as e:
    return f"错误：解析天气数据失败 - {e}"
```
